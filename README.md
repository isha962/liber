# Liber

Liber is a mobile-first reading tracker inspired by activity apps. Users can add books, log reading sessions, and export a Strava-style reading story as a PNG.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Vitest + Testing Library
- `html-to-image` for share export

## Run locally

```bash
cd /Users/ishakm/Desktop/liber
npm install
npm run dev
```

## Verify

```bash
npm test
npm run build
```
